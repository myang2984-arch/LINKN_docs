## 前言

感谢您使用 Linkn App。我们致力于保护您的个人隐私和数据安全。本隐私政策详细说明了我们如何收集、使用、存储和保护您的个人信息。

通过使用 Linkn App，您同意本隐私政策中描述的数据收集和使用方式。

---

## 1. 我们收集的信息

我们通过 Firebase Authentication 服务收集以下信息，用于账号管理和提供核心同步服务：

### 1.1 用户身份信息
- **邮箱地址**：来自 Google 或 Apple 登录的邮箱地址
- **用户标识**：唯一的 Firebase User ID (UID)
- **登录时间**：账户创建和最后登录时间

### 1.2 Notion 集成信息
- **Notion 授权令牌 (Token)**：您授权我们用于访问您的 Notion 数据库的加密 Token
- **Notion 数据库 ID**：您选择同步的 Notion 数据库标识符
- 此 Token 仅存储在 Firebase Firestore 中，并用于 Cloud Functions 执行同步

### 1.3 链接元数据
- **URL**：您捕获或添加的网页链接地址
- **网页标题和描述**：从 URL 自动提取的元数据
- **状态标签**：To Read、On Hold、Completed 等阅读状态
- **自定义标签**：您添加的分类标签
- **时间戳**：链接创建和最后编辑时间

### 1.4 设备信息
- **设备类型**：iOS、Android 或桌面平台信息（仅用于调试和优化体验）
- **应用版本**：用于兼容性检查

### 1.5 使用数据（可选）
- **崩溃报告**：应用崩溃时的匿名技术信息（需用户同意）
- **性能数据**：应用加载时间和响应速度（匿名）

## 2. 信息的使用

我们仅将收集的信息用于以下目的：

### 2.1 账户管理
- 验证您的身份并管理您的 Linkn App 账户
- 在多设备间同步您的账户信息

### 2.2 提供核心服务
- 启用将您的链接同步到 Notion 数据库的功能
- 在本地设备、Firebase Cloud 和 Notion 之间实现三端数据同步
- 保存您的链接、标签和阅读状态

### 2.3 AI 预处理（付费功能）
- 使用您的链接元数据调用 Google Gemini API
- 提供智能标签建议和内容摘要服务
- 此功能为可选付费功能，需要您的明确同意

### 2.4 客户支持
- 用于回复您的查询和解决技术问题
- 诊断和修复应用错误

### 2.5 服务改进
- 分析应用使用模式以改进用户体验（仅使用匿名聚合数据）
- 开发新功能和优化现有功能

## 3. 信息共享与第三方服务

### 3.1 我们使用的第三方服务

#### Google Firebase
- **用途**：身份认证、云端数据存储、实时同步
- **数据**：用户 UID、邮箱、链接元数据、Notion Token（加密）
- **隐私政策**：[Firebase 隐私政策](https://firebase.google.com/support/privacy)

#### Notion API
- **用途**：将链接同步到您的 Notion 数据库
- **数据**：链接 URL、标题、描述、标签
- **隐私政策**：[Notion 隐私政策](https://www.notion.so/Privacy-Policy-3468d120cf614d4c9014c09f6adc9091)

#### Google Gemini API（可选付费功能）
- **用途**：AI 智能摘要和标签建议
- **数据**：链接 URL 和元数据（不包含用户身份信息）
- **隐私政策**：[Google AI 隐私政策](https://policies.google.com/privacy)

#### Apple Sign In（iOS 用户）
- **用途**：身份认证
- **数据**：邮箱地址或匿名邮箱（由用户选择）
- **隐私政策**：[Apple 隐私政策](https://www.apple.com/legal/privacy/)

### 3.2 我们不会做的事
- 我们**不会**出售您的个人数据给第三方
- 我们**不会**将您的数据用于广告推广
- 我们**不会**在未经您同意的情况下与第三方共享您的信息（法律要求除外）

## 4. 信息存储与安全

### 4.1 存储位置
- **云端存储**：所有用户数据均存储在 Google Firebase Cloud Firestore 数据库中
- **数据中心**：位于 Google Cloud 的美国区域服务器
- **本地存储**：链接元数据在设备上使用 SQLite 加密存储

### 4.2 安全措施
- **加密传输**：所有数据传输使用 HTTPS/TLS 加密
- **Token 加密**：Notion Token 在存储前经过加密处理
- **访问控制**：Firestore 安全规则确保只有您本人可以访问自己的数据
- **身份验证**：基于 Firebase Authentication 的多因素身份验证支持

### 4.3 数据保留期限
- **活跃账户**：数据将持续保留，直到您删除账户或数据
- **非活跃账户**：超过 2 年未登录的账户数据将被自动删除（提前 30 天邮件通知）

## 6. 数据保留与删除

### 6.1 账户删除流程
如果您选择删除 Linkn App 账户：

1. **即时删除**：Firebase 认证账户信息和 Firestore 用户数据立即标记为删除
2. **30 天缓冲期**：数据将在 30 天内完全清除（防止误操作）
3. **永久删除**：30 天后所有数据不可恢复

### 6.2 删除范围
- **包含**：邮箱地址、Notion Token、所有链接元数据、标签、阅读状态
- **不包含**：已同步到您 Notion 数据库中的数据（您需要自行管理）

### 6.3 备份数据
- 在删除账户前，建议您先导出所需数据
- 删除后的数据无法恢复

## 7. 儿童隐私保护

Linkn App 不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。如果您是家长或监护人，发现您的孩子在未经您同意的情况下向我们提供了个人信息，请联系我们，我们将立即删除相关信息。

**符合 COPPA（儿童在线隐私保护法）规定。**

Linkn App 使用的 Firebase 服务器位于美国。如果您位于欧洲经济区（EEA）、英国或其他地区，您的数据可能会被传输到您所在司法管辖区之外的地区。

我们确保此类传输符合适用的数据保护法律，并采取适当的保护措施（如标准合同条款）。
